## 二、🎯 智元面试（目标，要最细准备）

### 风格特征
- **面试官会顺着项目细节深挖**，对"只背八股、没做过项目"非常敏感
- 会问竞品差异（Q9）、智元产品了解（Q10）、产线落地细节（Q4）
- 场景题多：给一个客户场景，让你现场设计方案和算ROI

### 高频问题预测
| 类别 | 问题 | 你的应答位置 |
|:---|:---|---:|
| 产品了解 | "你怎么看智元的产品矩阵？" | 智元产品矩阵速查表 |
| 竞品 | "智元 vs 宇树/Figure 区别？" | 竞品对比矩阵 |
| 场景 | "3C上下料方案怎么设计？" | 方案B制造业 |
| 数据 | "ROI怎么算的？" | ROI计算器+场景A/B |
| 技术 | "VLA为什么在制造业不够？" | 四级兜底+Eye-in-Hand |
| 追问 | "Sim2Real gap怎么办？" | Genie Sim+冷启动路径 |
| 文化 | "你做过最落地的项目？" | STAR故事 |

### 面试策略
1. **提前说ROI数据**：所有回答先拍数据（G2单台¥57万/4台¥228万/IRR 78.2%）
2. **引用原话**：面试中带出姚院/张总的原话（"Beyond Algorithms"、"要上了产线就是100%成功"）
3. **展现经验迁移**：每次说完技术方案，都加上一句"这个和我之前做X的经验一样"
4. **准备反问**：面试最后问"贵司在3C制造之外的行业拓展计划是什么样的？"

## 三、🔵 宇树面试（练手，第一站）

### 风格特征
- **据已入职者反馈"面试较水"**（小红书面经：月薪38K的AI产品岗面试很水）
- 更看重**产品思维、行业认知、学习能力**，不深究技术细节
- 面试官可能是产品负责人，非技术出身

### 高频问题预测
| 问题 | 应答思路 |
|:---|:---|
| "你怎么看宇树的产品线？" | 四足Go系列+人形H1/G1+过会动态 |
| "具身智能的产品思维？" | 先场景后产品，客户要的是降本不是机器人 |
| "做过什么AI产品？" | 产线Agent产品，0→1经验 |
| "竞品分析" | 智元/星动/宇树差异 |
| "对机器人行业理解" | 市场规模+出货量+成本曲线 |

### 面试策略
1. 不需要太多技术细节，多说自己对**产品和市场的理解**
2. 强调你的工业背景+AI产品经验
3. 把宇树当作"练手"——投递时间最早、面试最简单、积累经验
4. **投递时间**：07.14，先投先面

## 四、🔵 星动纪元面试（练手，第二站）

### 风格特征
- **清华系创业团队**，面试官可能偏科研背景
- 关注技术原理理解（VLA/RL/Sim2Real）
- 物流场景是他们的核心，可能会深问仓储分拣

### 高频问题预测
| 问题 | 应答思路 |
|:---|:---|
| "VLA和传统方案区别？" | 12概念第6个 |
| "物流仓储方案怎么设计？" | 场景A仓储方案 |
| "你怎么看星动 vs 智元的方案差异？" | 星动偏物流+科研、智元偏制造+工业 |
| "灵巧手的选型？" | XHAND 12dof vs G2夹爪 |

### 面试策略
1. 强调你也做过**AGV仓储盘点**，与星动物流场景直接对口
2. 展现对**VLA技术原理**的理解（12概念精讲就够了）
3. 这是你的**经验最对口**的练手公司

## 五、🔵 银河通用面试（练手，第三站）

### 风格特征
- 北京大学背景（创始人王鹤教授），注重**技术广度+商业洞察**
- 场景是零售/便利店+工业重载（Galbot S1进宁德/博世/丰田）
- 会问你怎么看待"具身智能在零售场景的落地"

### 高频问题预测
| 问题 | 应答思路 |
|:---|:---|
| "机器人进便利店怎么做？" | 银河太空舱案例引用 |
| "AstraBrain和GO-2比较？" | 都做端到端VLA，同代技术 |
| "重载场景怎么做？" | 双臂50kg负载，适合制造 |
| "你对零售场景ROI怎么看？" | 便利店的场景ROI分析 |

### 面试策略
1. 多说你对**多场景**的理解（不局限在3C制造）
2. 展示你对"具身智能在不同行业的ROI差异"有系统思考
3. 银河通用的场景（零售/便利店）和你的歌尔经验有差距，但可以作为**转型思路的证明**
