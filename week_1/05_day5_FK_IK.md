# Day 5 · FK/IK 概念 + 仿真工具概览（06.21）

## 学习目标

不推公式，理解 Forward Kinematics 和 Inverse Kinematics 各自解决什么问题。

## 知识框架

| | FK（正向运动学） | IK（逆运动学） |
|:---|:---|:---|
| **问题** | 已知关节角度 → 末端在哪？ | 已知末端目标位置 → 关节该转多少？ |
| **难度** | 简单，唯一解 | 多个解/无解，需优化 |
| **类比** | 你知道自己手臂三个角度，手在哪？ | 你看到桌上的杯子，手臂要怎么够到？ |

```
FK: θ₁,θ₂,θ₃... → (x, y, z, roll, pitch, yaw)
IK: (x, y, z, roll, pitch, yaw) → θ₁,θ₂,θ₃...
```
