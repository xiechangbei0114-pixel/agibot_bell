# Day 1 · 机器人分类（06.17）

## 学习目标

能把工业 / 协作 / 人形 / AGV 这四类区分清楚，标注自由度、精度、典型应用场景。

## 知识框架

```
机器人
├─ 工业机器人（传统六轴）
│   例：发那科、ABB
│   特点：高精度、高速度、需安全围栏
│   典型场景：焊接、喷涂、搬运
│   精度：±0.02mm ~ ±0.1mm
│
├─ 协作机器人（Cobot）
│   例：优傲 UR、智元灵犀 X1
│   特点：力控、人机协作、部署灵活
│   典型场景：3C 上下料、装配、检测
│   精度：±0.1mm ~ ±0.5mm
│
├─ 人形机器人
│   例：智元远征 A2、宇树 G1、Tesla Optimus
│   特点：双足/双臂、通用性高
│   典型场景：家庭服务、柔性制造
│
└─ AGV/AMR（移动机器人）
    例：极智嘉、海康
    特点：轮式底盘、2D 导航
    典型场景：仓储搬运、产线配送
```
