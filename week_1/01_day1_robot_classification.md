## 1. 机器人四大分类

### 1.1 工业机器人（Industrial Robot）

**定义**：固定底座的多关节机械臂，在高重复性、高精度场景中替代人工。

| 特征 | 数值/说明 |
|:---|:---|
| 自由度 | 通常 4–6 DOF（SCARA 4 轴，六轴臂 6 DOF） |
| 重复定位精度 | ±0.02 mm ~ ±0.1 mm（发那科 M-10iA: ±0.03 mm） |
| 负载 | 3 kg ~ 2300 kg（KUKA Titan） |
| 工作半径 | 0.5 m ~ 3.9 m |
| 安全要求 | **需安全围栏**，人机物理隔离 |

**典型品牌**：发那科（FANUC）、库卡（KUKA）、ABB、安川（Yaskawa）

**典型场景**：
- 汽车焊接：点焊/弧焊，节拍 < 60s
- 喷涂：防爆环境，轨迹精度要求高
- 搬运/码垛：重载（> 100 kg）
- 打磨/去毛刺：需要力控补偿


### 1.2 协作机器人（Cobot）

**定义**：设计为与人近距离协同工作的机械臂，配备力/力矩传感器实现安全交互。

| 特征 | 数值/说明 |
|:---|:---|
| 自由度 | 通常 6–7 DOF |
| 重复定位精度 | ±0.1 mm ~ ±0.5 mm（优傲 UR5e: ±0.03 mm） |
| 负载 | 3 kg ~ 35 kg（UR30: 30 kg；Franka Emika: 3 kg） |
| 工作半径 | 0.5 m ~ 1.9 m |
| 安全要求 | 力限制（ISO/TS 15066），无需围栏 |

**典型品牌**：优傲（Universal Robots）、Franka Emika、节卡（JAKA）、遨博（AUBO）、**智元灵犀 X1**

**典型场景**：
- 3C 上下料：工件轻、换线频繁
- 装配：人机协作拧螺丝/插装
- 检测/测量：手持工件配合视觉检测
- 实验室移液


### 1.3 人形机器人（Humanoid Robot）

**定义**：具有头部、躯干、双臂、双足的类人形态机器人，目标是实现通用任务能力。

| 特征 | 数值/说明 |
|:---|:---|
| 自由度 | **40–200+ DOF**（智元远征 A2: 49 DOF；Tesla Optimus Gen 2: 42 DOF） |
| 末端精度 | 厘米级（vs 工业臂毫米级），仍在快速迭代 |
| 负载 | 单臂 3–5 kg（远征 A2）、Optimus 单手 8 kg |
| 续航 | 1–4 小时 |
| 成熟度 | 实验室→小批量产过渡期 |

**典型厂商**：
- **智元（AGIBOT）**：远征 A2（49 DOF，全身 40+ 个执行器）、灵犀 X1/X2（全栈开源）
- 宇树（Unitree）：G1（23~43 DOF）、H1
- Tesla：Optimus Gen 2（42 DOF，灵巧手 22 DOF）
- Figure AI：Figure 02
- 1X Technologies：NEO

**典型场景（当前/近期）**：
- 3C 产品上下料、装配
- 物料搬运、料箱拆码垛（配合 AMR）
- 家庭服务（长期愿景）

> 💡 **关键认知**：2024–2025 年是人形从「演示」到「工厂试点」的转折点。智元 2024 年 8 月发布 5 款商用人形机器人，2025 年初已在 3C 产线小批量部署。


### 1.4 AGV / AMR（移动机器人）

| | AGV | AMR |
|:---|:---|:---|
| **导航方式** | 固定路径（磁条、二维码） | 自主导航（SLAM、LiDAR） |
| **避障能力** | 只能停车等待 | 动态规划绕行 |
| **部署成本** | 低设备 + 高基建（埋线、贴码） | 高设备 + 低基建 |
| **产线变更** | 需重做地面标记 | 软件更新地图即可 |
| **负载** | 可达 2,000+ kg | 通常 < 1,500 kg |
| **典型厂商** | 海康、极智嘉 | MiR、OTTO、海康（AMR 线） |

**应用场景**：
- 仓储搬运（亚马逊 Kiva 类 AMR）
- 产线配送（SMT 物料→产线）
- 配合机械臂 = **复合机器人**（AMR + Cobot，移动操作）

## 3. 选型逻辑（面试必问）

```
高精度、大批量、固定工序 → 传统工业机器人
小批量、多品种、人机混线 → 协作机器人
移动搬运 + 简单操作    → AMR + Cobot（复合）
通用性要求高、多工位   → 人形机器人（未来 2-3 年）
```

## 4. 自检答案

> **Q: 智元的远征 A2 和灵犀 X 分别属于哪类？为什么 3C 产线更可能用灵犀 X？**

**A:**
- **远征 A2**：人形机器人（49 DOF，1.7 m 高），定位通用任务。
- **灵犀 X1/X2**：协作机器人（单臂/双臂），全栈开源、部署灵活。

3C 产线为什么是灵犀 X？
1. **精度**：3C 装配间隙 < 0.5 mm，人形目前厘米级精度不满足
2. **占地**：产线工位紧凑，灵犀 X 固定底座占位小；人形双足需要更大空间
3. **成本**：灵犀 X 价格远低于远征 A2
4. **成熟度**：协作臂技术已成熟，人形还在 pilot 阶段
