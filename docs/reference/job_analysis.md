## 二、岗位 A：资深解决方案经理（工业）⭐ 推荐

### JD 原文

**职位描述：**
- 深度理解工业客户生产场景及痛点：深入工厂、仓储、制造等环境，分析生产流程、物流工段、设备协作模式，提炼自动化/智能化需求，输出需求文档及行业解决方案
- 参与产品策略与行业化能力建设：结合工业场景特性（如交互接口、调度集成、安全规范），参与制定产品行业策略，推动标准化与模块化方案建立
- 项目推进与 PoC 落地：负责工业行业的方案验证项目（PoC/试点）的排期、资源协调、现场落地及风险排查，保障项目按期交付效果
- 跨团队沟通与资源协同：与销售、研发、算法、硬件、交付等团队紧密合作，确保客户场景需求与产品技术能力有效匹配
- 行业洞察与竞争分析：持续跟踪工业机器人、自动化设备、AGV/AMR、MES/WMS 等系统的发展趋势及竞品动态，沉淀行业方法论，形成可复制的行业解决方案
- 客户交流与方案支持：面向客户进行解决方案宣讲、技术答疑、标书支持、价值呈现，提升工业客户的成单率与落地成功率

**职位要求：**
- 本科及以上学历，具备工业自动化、智能制造、机器人、AGV/AMR、设备管理等相关行业经验优先
- **有 0-1 解决方案打造或工业客户交付经验者加分**
- 了解制造业产线流程（如电子制造、汽车、零部件加工等），熟悉工业协议（Modbus、OPC-UA 等）或工业安全规范者优先
- 具备 ToB 项目管理能力，熟悉项目流程，掌握至少一种 PM 工具（Jira/飞书项目/禅道）
- 具备一定硬件机电知识、网络与系统集成知识，能与研发团队深度沟通需求与可行性
- 逻辑清晰、沟通能力强，具备行业热情、抗压能力和跨团队协同能力，可接受出差

## 三、岗位 B：应用工程师（部署态）⭐ 隐藏彩蛋

### JD 原文

**职位描述：**
围绕具身机器人"算法-模型-应用-本体-末端执行"全链路，负责对接部署态项目与伙伴之间的技术沟通：

1. **了解工业部署态项目核心链路**
   - 对具身机器人在工业场景提供有参考价值的解决方案计划，协助输出包含硬件选型、软件架构、部署流程的完整方案文档
   - 协助部署态项目落地，协调算法、硬件、测试团队解决部署中的兼容性、稳定性问题
   - 制定部署态项目落地后的运维方案

2. **对算法与模型协同的熟悉掌握**
   - 理解运动控制、感知决策、人机交互等核心算法逻辑，将算法能力转化为可落地的解决方案功能
   - 参与机器人模型优化，提出模型轻量化、实时性提升建议（如模型压缩、推理加速方案）
   - **验证算法与模型在实际场景中的效果，收集异常数据并反馈给算法团队，推动迭代优化（如定位精度、抓取成功率提升）**

3. **前期配合项目组导入需求**
   - 深入调研目标行业的具身机器人应用需求，输出需求分析报告和场景化解决方案
   - 负责应用功能的设计与落地，如工业场景的 **"物料抓取-搬运-分拣"全流程应用**
   - **解决本体与末端执行器的兼容性问题，如机械接口匹配、控制信号同步、力控参数校准等**

4. **跨团队协作与文档输出**
   - 协调算法、硬件、测试、产品团队推进项目
   - 输出标准化文档：解决方案手册、部署指南、运维手册、故障排查手册

**职位要求：**
1. 熟悉具身机器人全链路技术逻辑，具备场景分析能力
2. 理解机器人核心算法：运动控制（轨迹规划、力控）、感知（视觉识别、SLAM）、决策（强化学习、行为树）
3. 了解深度学习模型：目标检测（YOLO、Faster R-CNN）、姿态估计
4. **熟悉机器人开发工具：ROS、MoveIt，能进行机器人运动调试**
5. 具备 Python/C++ 基础编程能力
6. **有工业场景具身机器人解决方案落地经验（如 3C 电子行业）优先**
7. 参与过开源具身机器人项目（如 Isaac Sim、ROS 2）优先
