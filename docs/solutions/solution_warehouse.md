# 📦 方案 A：仓储物流 — 标准品建基线，非标打溢价

> 对标企业：Amazon Vulcan、Agility Digit
> 场景价值：具身智能目前 ROI 最清晰、落地最快的赛道
> 面试锚点：你的 AGV 仓储盘点项目经验

---

## 一、场景定义

### 典型工位

```
来料传送带 ──▶ 扫码/称重 ──▶ 机械臂抓取 ──▶ 放入目标容器
                    ↑                           │
                WMS 系统(知道来的是什么、放哪)     │
                                                  ▼
                                     破损/异常品分流回收
```

### 关键参数

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| SKU 种类 | **200-2000+** | 远多于制造业，VLA 的泛化能力在这里最有价值 |
| 精度要求 | ± **10-20 mm** | 放得进料箱即可，不需要 0.5mm |
| 节拍 | **8-15 s / 件** | 比制造业慢，时间裕度大 |
| 来料一致性 | ±50-100mm | 传送带定位极差，VLA 泛化刚好擅长 |
| 工作场景 | 结构化 | 货架/传送带布局固定，光照稳定 |

### 为什么 VLA 在这里是最优解

```
精度低、SKU 多、来料乱
     ↓
传统视觉（YOLO + PnP + IK）做不了：
  2000 种 SKU → YOLO 需要标注 2000 个类别 → 维护成本爆炸
  来料偏差 ±10cm → 固定视觉基线解决不了

VLA 不做显式分类，不依赖精确标定：
  图片 → 直接出动作 → 靠泛化能力覆盖所有 SKU
```

## 三、技术要点

### 3.1 为什么仓储场景 VLA 成功率最高？

三个原因叠加：

| 因素 | 制造业 | 仓储 | 农业 |
|:---|:---:|:---:|:---:|
| 精度要求 | ±0.5mm | ±10mm | ±20-50mm |
| 光照控制 | 稳定 | 稳定 | 室外变化大 |
| SKU 变化 | 慢（周级） | 快（天级） | 季节性 |
| **VLA 适合度** | ★★★ | **★★★★★** | ★★★★ |

仓储是三者中 **VLA 优势最大、劣势最小**的场景。

### 3.2 冷启动策略（关键！面试必问）

仓库刚部署时 VLA 没数据，怎么开始？

```
阶段 1 (第 1 周)：规则兜底
  只有第二层视觉运行（YOLO + 固定位姿）
  同时遥操作采集 VLA 训练数据
  精度够了，但 SKU 覆盖率低

阶段 2 (第 2-3 周)：混合模式
  见过的 SKU → VLA 处理
  没见过的 SKU → 回退到视觉
  采集到的数据积累到飞轮中

阶段 3 (第 4 周+)：VLA 主力
  覆盖度 >80%
  VLA 处理大部分情况
  视觉/力控只做兜底
```

**面试话术：**
> "仓库冷启动分三步：第一周先跑传统视觉兜底，同时遥操作采 500 条 demo；第二周混合模式，见过的 VLA 处理、没见过的回退视觉；第四周后 VLA 覆盖 80%+ 的主力场景。这套飞轮策略跟我的 Agent 产品冷启动一模一样——先上规则保底，边跑边积累数据，模型慢慢变成主力。"

### 3.3 Amazon Vulcan 的启示

| 启示 | 对你的意义 |
|:---|:---|
| **固定工位优于移动** | 先做固定工位抓取，别一上来追人形 |
| **标准品建基线** | 先把 20% 的标准品用 VLA 跑通，证明 ROI |
| **非标打溢价** | 非标品人工处理成本高，VLA 泛化解决 = 客户愿意多付钱 |
| **节拍不是第一** | 仓库容忍 8-15s 节拍，远低于制造业的 3s |

## 四、硬件配置与成本估算

| 组件 | 规格 | 估算成本 |
|:---|:---|:---:|
| 机械臂 | 智元 GO-2 / 协作臂 (6-7 DOF) | ¥50,000-150,000 |
| 夹爪 | 自适应二指夹爪 (Robotiq 2F-85 等) | ¥10,000-25,000 |
| 头顶相机 | Intel RealSense D435 / 海康 RGB | ¥2,000-5,000 |
| 手腕相机 | 小型工业相机 (EIS 等) | ¥1,000-3,000 |
| 工控机 | RTX 3060+ / Jetson Orin | ¥8,000-15,000 |
| **单工位合计** | | **¥70,000-200,000** |

### ROI 计算

```
假设：
  人工成本: ¥8,000/月/人 (三班倒 × 2人 = ¥48,000/月)
  机械臂: 替代 1.5 人 = ¥72,000/月 节省
  单工位投资: ¥150,000

投资回收期: 150,000 / 72,000 ≈ 2.1 个月
年化 ROI: (72,000 × 12 - 维护成本) / 150,000 ≈ 500%
```

### 面试话术（完整版）

> **"仓储物流是我最看好的 VLA 落地场景。原因有三：**
>
> **第一，精度要求低——±10mm 就够了，这是 VLA 现有的能力范围内。VLA 不需要像制造业那样做到 0.5mm，它最大的弱点在这里恰恰不是弱点。**
>
> **第二，SKU 多正是 VLA 的价值所在——2000 种商品，传统视觉不可能每个都标，VLA 看一眼就能抓，这种泛化能力在仓库里是刚需。**
>
> **第三，ROI 清晰——单工位替代 1.5 人，投资回收期 2-3 个月。Amazon 已经证明了这条路走得通。**
>
> **我在歌尔做 AGV 仓储盘点时，最大的体会是：标准品建基线、非标打溢价。VLA 在仓库里一样——先把 80% 的标准件跑通，让客户看到 ROI，再逐步覆盖非标品。这个递进策略是具身智能冷启动的核心思路。"**

## 六、风险与应对

| 风险 | 概率 | 应对 |
|:---|:---:|:---|
| VLA 泛化不稳定，偶发抓空 | 高 | 四级兜底，最后一层人工 30s 解决 |
| 新品类/新包装出现 | 中 | 飞轮机制，每周补采数据微调 |
| 节拍跟不上峰值需求 | 低 | 双工位并联，峰值时两台一起跑 |
| 客户担心 VLA 不可解释 | 中 | Action CoT 可追溯 + 兜底日志 |
