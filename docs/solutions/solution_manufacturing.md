> 对标企业：Figure × BMW、Hyundai Metaplant
> 场景价值：人形机器人预期最高的赛道，也是安全性要求最高的赛道
> 面试锚点：你的歌尔 3C 产线经验

## 一、场景定义

### 典型工位

```
料盘来料 ──▶ 机械臂抓取 ──▶ 治具定位 ──▶ 加工/测试 ──▶ 取回 ──▶ 分拣良品/不良品
```

### 关键参数

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 精度要求 | **±0.3-0.5 mm** | 这是制造业最大的门槛 |
| 节拍 | **≤ 3-8 s / 件** | 比仓储快一倍以上 |
| SKU 种类 | **5-50 / 天** | 比仓储少很多，换产频繁 |
| 来料一致性 | **±2-5 mm** | 工装/夹具定位 |
| 安全标准 | **ISO 10218 / ISO 91469** | 合规是硬门槛 |

### 为什么制造业最难但最值得做

```
精度 ±0.5mm ← 这是 VLA 当前做不到的
节拍 3s     ← VLA 推理延迟 300-500ms 占了 15% 的时间
安全合规   ← 机器人伤了人不是赔钱的问题

但：
客单价高         ← 一条产线改造几百万起
客户粘性大       ← 一旦接入不会轻易换
竞争壁垒高       ← 能做 ±0.5mm 的公司没几家
```

## 三、技术要点

### 3.1 精度是怎么做到的？（面试核心问题）

这是你面试时被问到最多的问题——**"VLA 精度达不到 0.5mm，你怎么解决？"**

答案不是"让 VLA 更精确"，而是 **"VLA 做粗定位，精定位交给传统视觉"**：

```
VLA 能给的精度:   ±10-20mm（从 1 米外看整个场景）
YOLO 能给的精度:  ±2-5mm（检测到工件中心）
Eye-in-Hand 能给的精度: ±0.3mm（近距离 + 模板匹配）

最终精度: ±0.3mm → 满足 ±0.5mm 要求
```

**关键：不是一张图搞定一切，是分两步走**

```
             VLA 看一眼 → 方向对了
                │
                ▼
     机械臂移到工件上方 20cm
                │
                ▼
     手腕相机拍清楚 → 模板匹配/PnP → 精确到 0.3mm
```

**面试话术：**
> "VLA 的精度问题是个伪命题。你不需要 VLA 输出 ±0.5mm，你只需要它告诉你'大概在那个方向'。真正到 0.5mm 精度是靠 Eye-in-Hand 手腕相机 + 近距离拍照实现的——距工件 20cm 时，1 像素只有 0.06mm，加上模板匹配，±0.3mm 是成熟的工业方案。VLA 的价值不是替代精定位，而是省掉换产时的重新示教。换一个品种不用停机 2 小时改程序，省下来的时间远比精定位的 200ms 值钱。"

### 3.2 节拍怎么保证？

制造业节拍 3-8s，而 VLA 推理就要 300-500ms，再加精定位 200-500ms，视觉+决策占了总时间的 20-30%。怎么办？

**策略 1：流水线预取（Pipeline）**
```
相机在机器人还在执行时就开始拍下一件
推理和运动并行 → 感知延迟几乎被隐藏
```

**策略 2：关键路径优化**
```
只对关键工位做精定位：
  ┌─ 上料（精度要求低）→ VLA 直接抓
  └─ 治具插拔（精度要求高）→ VLA + 精定位
```

**策略 3：缓存已知品种**
```
同一个 SKU 的 VLA 推理结果缓存
下次遇到直接取 → 节省 300-500ms
```

### 3.3 安全标准（制造业独有的硬门槛）

| 标准 | 要求 | 对你的影响 |
|:---|:---|:---|
| ISO 10218-1/2 | 工业机器人安全要求 | 速度限制、力限制、安全距离 |
| ISO 91469 | 人形机器人安全标准（制定中） | 暂时无法规，按工业机器人标准做 |
| ISO 13849-1 | 控制系统安全 | 急停电路需独立于主控 |
| ISO 13855 | 安全距离计算 | 机器人速度越快，安全距离越大 |

**制造业部署的安全基线：**

```
1. 围栏隔离（初期）
   物理围栏 + 光栅 + 急停
   机器人速度不限，效率最大化
   
2. 人机协作（成熟期）
   去掉物理围栏
   力限制 < 150N，速度 < 250mm/s
   需要安全级认证
```

```

  改造一条上下料产线: 4 个工位 × ¥200,000 = ¥800,000
  替代人工: 每个工位 1.5 人，4 工位 = 6 人

投资回收期: 800,000 / 48,000 ≈ 16.7 个月
年化 ROI: ~70%（比仓储低，但客户粘性更高）
```

## 五、你与这个场景的缝合点

### 你的经验缝合（这是你最强的场景）

```
你在歌尔 3 年的产线经验
    │
    ├── 你懂 3C 上下料的精度要求（±0.5mm）
    ├── 你懂 MES / PLC 怎么对接
    ├── 你懂换产停机的成本有多高（1 小时 = ¥XX 万）
    ├── 你懂安全规范（光栅、急停、安全距离）
    └── 你懂产线上的"异常"是怎么管理的
                    │
                    ▼
VLA 在制造业的核心价值不是替代精度，而是省掉换产时间
    → 这件事你最清楚，因为你在产线上真实经历过
```

### 面试话术（完整版）

> **"制造业是具身智能最难但最有价值的场景。为什么难？三个硬指标：精度 ±0.5mm、节拍 3-8s、安全合规。VLA 一个都做不到——但这不意味着 VLA 在制造业没用。**
>
> **VLA 的价值不在精度，而在换产。我在歌尔的时候，换一个品种要停机 1-2 小时重新示教，一天换 5 个品种就是 5-10 小时停机时间。VLA 换产时不用停机——看一眼新工件就能抓。精度交给 Eye-in-Hand 手腕相机做到 ±0.3mm。**
>
> **我们的方案是：VLA 做粗定位 + 引导机械臂靠近，手腕相机做精定位 + 模板匹配保证 ±0.3mm 精度，力控做最后兜底。这套组合拳既拿到了 VLA 的泛化能力，又满足了制造业的精度要求。**
>
> **安全方面，初期走物理围栏隔离，等标准完善后逐步放开到人机协作。我在歌尔管过安全评审，这一块我懂。"**

## 六、风险与应对

| 风险 | 概率 | 应对 |
|:---|:---:|:---|
| 精度达不到客户要求 | 中 | Eye-in-Hand 双保险，可视化精度报告 |
| 节拍不够快 | 中 | 流水线预取 + 关键路径优化 |
| 安全认证周期长 | 高 | 先走围栏方案，安全认证与功能开发并行 |
| 客户不信任 AI | 高 | Action CoT 可追溯 + 每步日志 + 分级兜底 |
| VLA 模型更新引入 regression | 中 | A/B 测试 + 灰度发布 + 回滚机制 |

## 七、演进路线

```
阶段 1 (第 1-2 月)：传统视觉 + 围栏
  YOLO + PnP + IK 跑通
  验证精度和节拍达标
  安全围栏隔离

阶段 2 (第 3-4 月)：VLA 辅助换产
  VLA 并行运行但不控制机器人
  验证 VLA 输出与精定位的一致性
  积累换产场景数据

阶段 3 (第 5-6 月)：VLA 主力
  VLA 做主推理，精定位做校验
  换产零停机
  开始测试去掉围栏

阶段 4 (第 7 月+)：人机协作
  拿到安全认证
  去掉物理围栏，只保留光栅
  节拍再优化
```
