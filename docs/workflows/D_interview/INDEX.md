## 📝 练习方法

```
每天 3 题，手机录音，自己听

目标: 3 分钟内讲完，不卡顿超过 3 秒
框架: 先讲商业收益 → 方案架构 → 产品选型 → 技术实现
不要背稿，记住框架和关键词，用自己的话讲
```
