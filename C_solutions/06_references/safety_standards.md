
## 二、逐条速查

### 🔴 ISO 10218 — 工业机器人安全（根本标准）

| 部分 | 内容 | 对应国标 |
|:---|:---|---:|
| **ISO 10218-1** | 机器人**本体设计**安全 — 固有安全设计、保护措施、使用信息 | GB 11291.1 |
| **ISO 10218-2** | 机器人**系统与集成**安全 — 针对集成商的应用场景 | GB 11291.2 |

**核心要求：**
- 控制系统的安全相关部件需达到 **PL=d**（性能等级d）
- 结构类别 **Category 3**（双通道架构，单点故障不丢失安全功能）
- 或 **SIL 2**，硬件容错等级1（IEC 62061）

**最新动态：** ISO 10218:2025 已发布，将 ISO/TS 15066（协作机器人技术规范）整合进来。

### 🟡 ISO 13849 — 控制系统安全相关部件（方法标准）

| 概念 | 说明 |
|:---|:---|
| **性能等级 PL** | a~e 共5级，**PL=d** 是工业机器人标配（PFHd 非常低） |
| **Category 3** | 双通道架构——单点故障不会导致安全功能丧失 |
| **MTTFd** | 平均无危险故障时间，需足够高 |
| **DC** | 诊断覆盖率，需足够高 |
| **CCF** | 共因失效防护，需有措施 |

### 🟢 ISO 13855 — 防护装置定位（距离标准）

**核心内容：**
- 安全光幕安装距离 = 接近速度 × 停止时间 + 侵入深度
- 安全激光扫描仪的监测区域设置方法
- 围栏/安全门的联锁位置设计

## 三、制造业部署的安全基线

### 阶段1：初期（围栏隔离）
```
✅ 物理围栏 + 安全光栅 + 急停按钮
✅ 机器人速度不限，效率最大化
✅ 只要通过安全门→光栅断开→急停，人在围栏外就安全
✅ 符合ISO 10218-2要求
```

### 阶段2：成熟期（人机协作，围栏可移除）
```
✅ 去掉物理围栏
✅ 力限制 < 150N
✅ 速度 < 250mm/s（慢速模式）
✅ 需要额外的安全级认证（ISO/TS 15066）
✅ 六维力传感器监控接触力
```

## 四、面试话术

**Q: "你的方案怎么保证安全合规？"**
> "分两步走：初期走物理围栏隔离——完全满足ISO 10218-2要求，光栅+急停+围栏，速度快不限，效率最大化。等客户信任建立、安全认证拿到之后，再逐步放开到人机协作——力控限制150N、速度250mm/s。我在歌尔管过安全评审，这一套流程我熟。"

**Q: "ISO 13849的PL是什么意思？"**
> "PL是性能等级，从a到e共五级。工业机器人要求PL=d——意味着每小时危险失效概率低于10的负7次方。结构上走Category 3双通道设计，一个通道失效另一个还能保证安全。ISO 10218对机器人本体和系统集成都引用了这个标准。"
