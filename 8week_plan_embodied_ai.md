# 🗺️ 8周具身智能学习路线图 v4.0

> **制定日期**：2026-06-17 | **目标**：智元方案岗，但不止于会用平台
> **核心思路**：学平台背后的原理，不学平台按钮的操作。学判断力，不学调参。

---

## 🎯 先搞清楚：你值钱在哪

```
智元的产品在 2026 年：

  Genie Studio（零代码平台）                    GO-2 模型
  ┌────────────────────────┐            ┌──────────────┐
  │ 拖拽 → 训练 → 部署      │            │ VLA 端到端    │
  │ 半小时学会              │            │ 成功率 ~92%   │
  └────────────────────────┘            └──────────────┘
         ↑                                    ↑
    这是工具，像 Photoshop              这是引擎，像 GPT-4
    会用 ≠ 值钱                         知道边界在哪 = 值钱

你的价值：
  不在「会用 Genie Studio 拖拽训练」
  而在「客户产线出了问题，你知道是改数据、改兜底、还是改物理设计」
```

---

## 📅 8周路线：三层能力

```
底层 ─ 知道原理（不写代码，但能判断平台在干什么）
中层 ─ 知道边界（平台什么时候会翻车，翻车了怎么办）
上层 ─ 知道场景（客户产线的真正约束是什么，怎么翻译成平台参数）
```

| 阶段 | 时间 | 层次 | 核心任务 |
|:---:|:---:|:---|:---|
| 🟢 一 | 第1-2周 | 底层原理 | 理解机器人是怎么被控制的（从示教器→ROS→VLA） |
| 🟡 二 | 第3周 | 中层边界 | 理解 VLA 模型的能力和局限 |
| 🟠 三 | 第4-6周 | 上层场景 | 工业产线场景分析 + 兜底设计 |
| 🔴 四 | 第7-8周 | 包装 | 简历重构 + 模拟面试 |
