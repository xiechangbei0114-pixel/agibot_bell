## 1. 仿真栈的层级关系

```
┌─────────────────────────────────────────┐
│  Genie Sim 3.0（智元）                   │  ← 你面试要提的
│  LLM 生成场景 / 开源数据集 / 评估基准     │
├─────────────────────────────────────────┤
│  Isaac Sim（NVIDIA）                     │  ← 底层引擎
│  光线追踪 / 物理引擎 / Omniverse 集成     │
├─────────────────────────────────────────┤
│  NVIDIA GPU + Jetson                     │  ← 硬件
└─────────────────────────────────────────┘
```

> **Genie Sim 基于 Isaac Sim，不是替代它。** 就像安卓基于 Linux 内核——用户看到的是安卓，底层跑的是 Linux。

---

## 2. 仿真解决了什么问题

| 没有仿真 | 有仿真 |
|:---|:---|
| 在真机上试错 → 撞坏了要修 | 仿真里随便撞，不花钱 |
| 采一次数据 = 人遥控一次 | 仿真自动生成百万级数据 |
| 换一个 SKU → 重新采数据 | 改仿真参数 → 自动生成新场景 |
| 只能在白天采数据 | 仿真 24 小时跑 |
| 真机训练怕撞人 | 仿真里无所谓 |

> 跟你训练 YOLO 时做数据增强一样——翻转、加噪声、调亮度。仿真就是机械臂的 "数据增强"。

## 3. RL vs 模仿学习：两种训练方式

```
强化学习（RL）                      模仿学习（IL）
─────────────────                  ─────────────────
让模型自己探索                      看人怎么做，跟着学

方法：                                 方法：
  在仿真里随机动 → 抓到 +1 分          人用遥操作演示 50 次
               → 撞到 -10 分          模型记下 (图片→动作) 的映射
               → 逐步学会最优策略

优点：                                 优点：
  ✅ 不需要人演示（便宜）               ✅ 不需要定义 reward（简单）
  ✅ 可能找到人想不到的策略              ✅ 数据质量高（人是最优的）
  ✅ 能探索 corner case

缺点：                                 缺点：
  ❌ reward 设计是玄学                  ❌ 人采数据贵
  ❌ 训练不稳定                        ❌ 只见过"正确"做法，没见过失败
  ❌ 可能学到 reward hacking           ❌ 换场景要重新采


两者的关系（跟CV类比）：

  RL      = 让模型自己从零开始探索  ≈  从随机权重开始训 ResNet
  模仿学习  = 让模型模仿人类专家      ≈  用预训练模型 fine-tune

工业界实际：
  AGIBOT G2、Figure 02、Atlas → RL + 仿真（因为数据便宜、能试错）
  学术界论文                      → 模仿学习（因为不需要定义 reward）
```

## 4. Sim2Real：最难的环节

```
仿真里训好的模型 → 放到真机上
           │
           ├─ 问题1：仿真里的光照是完美的，真机上有反光
           ├─ 问题2：仿真里的摩擦力恒定，真机上工件有油
           ├─ 问题3：仿真里没有延迟，真机上通信有 15ms 延迟
           └─ 问题4：仿真里的物体位置精确，真机上 ±3mm 偏移

解决方法（这三招面试要能讲）：

  ① 域随机化（Domain Randomization）
     仿真里故意随机变光照/纹理/物理参数
     → 训出来的模型对真实变化不敏感
     → 就像 YOLO 训练时加的数据增强

  ② 系统辨识（System Identification）
     先测真机的真实物理参数
     → 回填到仿真里
     → 仿真越准，迁移越容易

  ③ 渐进式部署
     先在简单的"类仿真"场景上线
     → 慢慢放开约束
     → 就是你歌尔项目的渐进式交付！
```

## 5. Genie Sim 3.0 对 Sim2Real 的贡献

| 特性 | 干什么 |
|:---|:---|
| LLM 自然语言生成场景 | 「给我一个 3C 产线工位」→ 自动搭好，不用手动建模 |
| 10,000+ 小时开源数据 | 省了你自己采数据的时间 |
| 仿真 vs 真实差距 < 10% | 自测指标，Sim2Real 可靠性可量化 |
| MetaCam 60 秒建模 | 拿手机环拍实物 → 自动生成仿真资产 |
