# Day 12 · 兜底设计方法论

> 你最大的差异化竞争力——工业方案岗的核心
> 🕐 预计用时：60 min

---

## 1. 为什么需要兜底

```
VLA 模型的现实：
  论文成功率：98.5%（LIBERO 基准）
  产线首次成功率：~85-92%（真实环境有光照/来料偏差）
  客户要求：>99.5%（产线不能停）

  差距 ~7-15% → 怎么填？
  答案不是「把模型训得更准」——而是「设计兜底」
```

> 你越南贴胶带的经验就是兜底思维——**用物理手段替代算法精度。**

---

## 2. 四级兜底金字塔

```
                    ┌─────────────┐
  成功率            │ VLA 泛化     │  ← 主力方案，处理 80% 正常情况
  85-92%            └──────┬──────┘
                           │ 失败
                    ┌──────▼──────┐
  失败 →            │ 视觉定位重试  │  ← YOLO+3D 精确找到工件
  累计 95%          └──────┬──────┘
                           │ 再失败
                    ┌──────▼──────┐
  再失败 →           │ 力控兜底     │  ← 慢速接近 + 力反馈
  累计 98%          └──────┬──────┘
                           │ 还失败
                    ┌──────▼──────┐
  还失败 →           │ 人工介入     │  ← 报警 + 产线不停
  累计 99.9%        └─────────────┘
```

## 3. 每层的具体策略

### 第一层：VLA 泛化

```
干什么：GO-2 模型看到工件 → 输出抓取动作
适用：正常光照、产品在预期范围内
翻车原因：
  - 来料偏离 > 5mm（模型没见过这么大的偏差）
  - 光照突然变化（上午有阳光，下午没有）
  - 新 SKU 跟训练数据里长得完全不一样
```

### 第二层：视觉定位重试

```
干什么：VLA 失败了 → 切到你的老本行 YOLO+3D
  → 精确算出工件 6D 位姿 → IK 求解 → 直接执行
适用：VLA 没泛化但工件还在视野内
关键设计：
  - 最多重试 2 次（不能无限循环，节拍不允许）
  - 重试时相机从不同角度拍（换个视角可能就找到了）
  - 这是你最擅长的——工业视觉！
```

### 第三层：力控兜底

```
干什么：视觉也找不到或定位不准 → 慢速逼近 + 力反馈
  → 夹爪慢慢靠近「大概位置」
  → 碰到东西 → 合拢
  → 不是「看到再抓」，是「碰到了就抓」
适用：工件偏了但还在物理范围内
成本：每个工位可能要多 1-2 秒
```

### 第四层：人工介入

```
干什么：以上全失败
  → 机械臂归位
  → MES 报警
  → 产线不停止（其他工位继续跑）
  → 人工 30 秒内过来处理

这跟你在歌尔设计 Agent 时一样——
不要假设模型 100% 对，要设计它失败后怎么办。
```

## 4. 跟你的经验缝合

| 兜底层 | 你的歌尔经验 | 
|:---|:---|
| VLA 泛化 | Agent 里的 LLM——泛化强，偶发幻觉 |
| 视觉定位重试 | **这是你的老本行**——YOLO + SuperPoints + 相机标定 |
| 力控兜底 | 越南贴胶带——物理手段替代算法精度 |
| 人工介入 | Agent 里"不确定的问题转人工"——同一个逻辑 |

---

## 5. 面试话术

> "具身智能工业落地的核心不是让 VLA 100% 准——那不可能。核心是分层兜底。VLA 做泛化抓取，视觉定位做精确纠偏，力控做最后一步的物理保障，人工做终极兜底。这跟我做 Agent 产品的架构一样——大模型做泛化，工作流做兜底，人工做最后一道关。我在歌尔做视觉检测时也是这个逻辑——SuperPoints 做不到 100%，剩下的靠物理手段和流程补。"
